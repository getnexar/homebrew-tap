# Homebrew Tap for Nexar CLI

This is the official Homebrew tap for the Nexar CLI tool.

## Installation

```bash
brew install getnexar/tap/nexar
```

## Usage

After installation:

```bash
# Login to Nexar Platform
nexar login

# List your apps
nexar apps

# Create a new app
nexar create my-app

# Deploy your code
nexar deploy
```

## Updating

```bash
brew upgrade nexar
```

## Uninstalling

```bash
brew uninstall nexar
```
